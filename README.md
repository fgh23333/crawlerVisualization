## 怎么安装

1. 使用git克隆项目

   ```bash
   git clone https://github.com/fgh23333/crawlerVisualization.git
   ```
2. 安装依赖

   ```bash
   npm install
   ```
3. 运行项目


```bash
  npm run serve
```
